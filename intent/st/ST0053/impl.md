## Notes

- `IN-AU-STYLE-003` (front-matter/objectives) deliberately stays in the author pack -- it is book/course information architecture, not shared prose hygiene. The author style tier is therefore a single rule (a legal numbering gap, 001/002/004/005 migrated out).
- WP02 tier placement: the D4 "content genuinely owns" list mixed mechanical (page meta, alt-text, link text) and judgment (voice/scannability, CTA, reading-level) rules. Placed each by check-type into style vs craft, matching the author pack's convention. The ratified D4 scope is unchanged; only the tiering is an as-built call.
- WP03: `critic-prose` resolves its discipline from the project's declared `languages` (author, content, or both), so one critic serves every prose discipline (D3). The `--lang author`/`--lang content` invocation override exists for a mixed project or an unambiguous target.
