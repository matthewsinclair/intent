# WP-07: Packaging & Delivery Assets

## Objective

Create deliverable artifacts for actual course delivery by a facilitator.

## Deliverables

- Course README with setup instructions
- Per-day handout materials
- Sample/exercise repo for practical sessions
- Enterprise-specific framing materials (ROI, governance, risk)
- Evaluation framework as standalone handout
- Reading list / resource guide

## Acceptance Criteria

- Facilitator can deliver the course using only these materials
- Enterprise materials are credible without being patronizing
- Evaluation framework works standalone outside course context
