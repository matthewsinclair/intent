# WP-03: Landscape Research

## Objective

Comprehensive survey of the agentic coding landscape. Produce landscape ACIs and an evaluation framework for tools/methodologies.

## Deliverables

- Landscape survey (`docs/course/landscape.md`) covering 7 dimensions: Anthropic guidance, tool landscape, plugins ecosystem, open source methodology, enterprise adoption, mental models, notable practitioners
- Tool comparison matrix
- Evaluation framework (`docs/course/evaluation-framework.md`)
- 8-12 landscape ACIs
- Reading list / resource guide

## Acceptance Criteria

- All 7 landscape dimensions have substantive coverage
- Evaluation framework is concrete enough to actually use
- Landscape ACIs follow standard ACI format
